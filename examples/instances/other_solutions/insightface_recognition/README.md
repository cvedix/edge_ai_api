# InsightFace Recognition Solution (ONNX)

Solution nhận diện khuôn mặt sử dụng InsightFace với model ONNX.

## Ví dụ

- `example_insightface_recognition.json` - InsightFace recognition với ONNX model

## Tham số

- `FACE_DETECTION_MODEL_PATH`: Đường dẫn đến YuNet face detection model
- `FACE_RECOGNITION_MODEL_PATH`: Đường dẫn đến InsightFace recognition model (ONNX)
